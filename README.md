## 📦 Installation

1. Copy the code from `src/MainModule.lua`
2. In Roblox Studio, create a **LocalScript** in `StarterPlayer > StarterPlayerScripts`
3. Paste the entire code
4. Run the game - The admin panel will appear automatically!

## 🎯 Usage

### Access the Panel
- The panel appears automatically when you join the game
- Click and drag the title bar to move it
- Click ✕ to close it

### Navigate Tabs
1. **📊 Dashboard** - View server stats, uptime, FPS, memory
2. **👥 Players** - Search and manage players
3. **🛡 Moderation** - Quick access to warn, kick, ban, freeze, etc.
4. **🖥 Server** - Lock/unlock, restart, shutdown server
5. **⭐ Staff** - View all ranks and permissions
6. **📋 Logs** - Search and filter all actions
7. **⚙ Settings** - Theme, UI scale, notifications, accessibility
